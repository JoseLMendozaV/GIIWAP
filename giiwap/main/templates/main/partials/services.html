Líneas de Investigación

Lo que estamos investigando

Desarrollo Web

Investigación web ágil con Django y tecnologías modernas.

Machine Learning

Es una rama de la inteligencia artificial donde las máquinas aprenden de los datos sin ser programadas explícitamente.

Aprendizaje Profundo

Subcampo del Machine Learning que utiliza redes neuronales profundas para simular el comportamiento del cerebro y reconocer patrones complejos.

Visión por Computadora

Campo tecnológico que permite a las máquinas interpretar y tomar decisiones basadas en información visual, emulando la visión humana.

Análisis de Datos

Proceso de inspeccionar, limpiar y transformar datos para descubrir información útil, conclusiones y apoyar la toma de decisiones.

Ciberseguridad

Es la práctica de proteger sistemas, redes y datos de ataques, daños o accesos no autorizados en el ámbito digital.
